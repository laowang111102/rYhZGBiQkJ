# 旅游管理系统 java815

## 项目概述

旅游管理系统是一款基于前后端分离架构的Web应用程序，旨在提供一站式的旅游服务解决方案，支持定制化开发。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK 17
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 登录模块

- 用户登录
- 用户注册
- 忘记密码

### 用户前台

- 首页
- 景点信息（预约）
- 旅游路线
- 景区酒店（预约）
- 旅游咨询
- 我的收藏
- 我的预定

### 后台管理

- 预约统计图
- 景点管理
- 线路管理
- 酒店管理
- 咨询管理
- 景点预约管理
- 酒店预约管理
- 轮播图管理
- 用户管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- 前端：现代浏览器
- 后端：Java运行环境（JRE）

## 目录结构

```
├── src
│   ├── main
│   │   ├── java
│   │   ├── resources
│   │   └── webapp
│   └── test

```

## 部署步骤

1. 准备开发环境（IDEA、JDK 17、Maven等）
2. 克隆代码到本地
3. 导入项目到IDEA
4. 配置数据库（Mysql）
5. 执行数据库初始化脚本
6. 编译项目
7. 部署到Web服务器

## 核心亮点

- 前后端分离，易于维护和扩展
- 基于SpringBoot，开发效率高
- 支持多种角色权限管理
- 可接受定制化开发，满足个性化需求
